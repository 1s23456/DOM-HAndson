Q2: Write the code to access element which is having tag as "h1".

               ANS: let second = document.getElementsByTagName("h1");
                         console.log(second);

Explanaton:- This code selects all the HTML elements with the tag name "h1" and assigns them to the variable second. The document.getElementsByTagName() method is used to retrieve all the elements with a specified tag name, in this case "h1".
